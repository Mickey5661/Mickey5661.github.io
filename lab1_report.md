## Задание 1. Обмен сообщениями по UDP

**Суть работы:**  
Создано клиент-серверное приложение, где обмен происходит по протоколу UDP. Клиент отправляет произвольное сообщение серверу, сервер отображает адрес отправителя и содержимое сообщения, после чего отвечает стандартным сообщением.

**Инструкция по запуску:**  
1. Запустить сервер:  
```bash
python3 udp_server.py
```  
2. Запустить клиент:  
```bash
python3 udp_client.py
```  
3. Ввести любое сообщение, например: `Hello, server`  
4. На сервере появится адрес и текст сообщения.  
5. Клиент получит ответ: `Hello, client`.

## Задание 2. TCP-сервер для вычисления гипотенузы

**Суть работы:**  
Сервер по протоколу TCP принимает от клиента два числа — длины катетов — и вычисляет длину гипотенузы, возвращая результат клиенту.

**Инструкция по запуску:**  
1. Запустить сервер:  
```bash
python3 server_pythagoras.py
```  
2. Запустить клиент:  
```bash
python3 client.py
```  
3. Ввести два числа, например: `3 4`  
4. Сервер вернёт результат: `Гипотенуза для катетов 3.0 и 4.0: 5.000`

## Задание 4. Многопользовательский чат с потоками (TCP + threading)

**Суть работы:**  
Сервер позволяет подключать несколько клиентов одновременно. Все сообщения, отправленные одним клиентом, пересылаются остальным участникам чата. Для параллельной работы подключений используются потоки.

**Инструкция по запуску:**  
1. Запустить сервер:  
```bash
python3 chat_server.py
```  
2. Запустить несколько клиентов в отдельных терминалах:  
```bash
python3 chat_client.py
```  
3. Обмениваться сообщениями. Все участники видят сообщения друг друга.  
4. Для выхода из чата использовать команду: `/quit`

## Задание 5. TCP сервер с журналом оценок (HTTP GET/POST)

**Суть работы:**  
Сервер хранит оценки по предметам и предоставляет их в виде HTML-страницы.  
- **GET** — отображает текущий журнал  
- **POST** — добавляет новую оценку

**Инструкция по запуску:**  
1. Запустить сервер:  
```bash
python3 journal_server.py
```  
2. Просмотр журнала через браузер:  
[http://localhost:8082](http://localhost:8082)  
3. Просмотр журнала через терминал:  
```bash
curl http://localhost:8082
```  
4. Добавление оценки через POST-запрос:  
```bash
curl -X POST -d "subj=Математика&grade=5" http://localhost:8082
```  
5. После отправки новой оценки она автоматически появляется в журнале на странице.
